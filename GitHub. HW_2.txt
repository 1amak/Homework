Создать новый репозиторий Branches
git clone https://github.com/1amak/Branches.git
cd Branches
1. На локальном репозитории сделать ветки для:
- Postman           git branch Postman
- Jmeter            git branch Jmeter
- CheckLists        git branch CheckLists
- Bug Reports       git branch Bug_Reports
- SQL               git branch SQL
- Charles           git branch Charles
- Mobile testing    git branch Mobile_Testing


2. Запушить все ветки на внешний репозиторий
 git push origin --all

3. В ветке Bag Reports сделать текстовый документ со структурой баг репорта
git checkout Bug_Reports
touch structure_BG.txt
vim structure_BG.txt
Insert

1) Summary;
2) Environment;
3) Steps to reproduce;
4) Attachment;
5) Severity;
6) Priority;
7) Actual result;
8) Expected result.

ESC
:wq

4. Запушить структуру багрепорта на внешний репозиторий
git add structure_BG.txt
git commit -m "Create structure_BG.txt"
git push --set-upstream origin Bug_Reports

5. Вмержить ветку Bug Reports в Main
git checkout main
git merge Bug_Reports

6. Запушить main на внешний репозиторий.
git commit -m "Create structure.txt"
git push

7. В ветке CheckLists набросать структуру чек листа.
git checkout CheckLists
touch checklists.txt
vim checklists.txt
Insert

1) Build;
2) Environment;
3) Tester;
4) Summary;
5) Status;
6) Comment.

ESC
:wq

8. Запушить структуру на внешний репозиторий
 git add checklists.txt
 git commit -m "Create checklists.txt"
 git push --set-upstream origin CheckLists

9. На внешнем репозитории сделать Pull Request ветки CheckLists в main

10. Синхронизировать Внешнюю и Локальную ветки Main. git checkout main, git fetch, git push
